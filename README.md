# Norah's Notes Podcast

A family-friendly podcast website featuring giggles, jokes, and imaginative stories told by a seven-year-old.

**Live site:** [https://norahtashner.com](https://norahtashner.com)

**Listen on Spotify:** [Norah's Notes Podcast](https://open.spotify.com/show/4VY57mUtnr7fqRtmN2j9Ra)

---

## Features

- Hero section with podcast artwork and subscribe button
- Episode listings with Spotify embeds
- About section introducing the show
- Dark/light theme toggle
- Fully responsive design
- SEO optimized with Open Graph tags

## Tech Stack

- React 19
- Vite
- CSS (no frameworks)
- Deployed to AWS S3 + CloudFront

## Development

```bash
npm install
npm run dev
```

Visit: http://localhost:5173

## Build & Deploy

```bash
npm run build
```

Deployed automatically via GitHub Actions on push to `main`.
